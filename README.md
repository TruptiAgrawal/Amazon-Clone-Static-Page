# Amazon Clone - Static E-commerce Page

A responsive Amazon homepage clone with interactive features built using HTML, CSS, and vanilla JavaScript.

## Features

✅ **Search Functionality** - Real-time product filtering by name  
✅ **Shopping Cart** - Add/remove items with persistent storage (localStorage)  
✅ **Category Filter** - Filter products by category (Beauty, Fashion, Accessories, Home & Office)  
✅ **Responsive Hamburger Menu** - Mobile-friendly navigation  
✅ **Wishlist Feature** - Save favorite products with heart icon toggle  
✅ **Responsive Design** - Works on desktop, tablet, and mobile devices  
✅ **Smooth Animations** - Hover effects and transitions  

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (ES6)
- Font Awesome Icons
- LocalStorage API

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. No build process or dependencies required!

## Features in Detail

### 1. Search Functionality
- Type in the search bar to filter products in real-time
- Case-insensitive search
- Searches through product names

### 2. Shopping Cart
- Click "Add to Cart" to add products
- View cart by clicking the cart icon
- Remove items from cart
- Displays total price
- Cart data persists using localStorage

### 3. Category Filter
- Filter products by category from the navigation panel
- Categories: All Products, Beauty, Fashion, Accessories, Home & Office
- Active category highlighting

### 4. Responsive Hamburger Menu
- Appears on mobile devices
- Slide-out menu with categories
- Overlay background

### 5. Wishlist
- Click heart icon to add/remove from wishlist
- Visual feedback with filled/outlined heart
- Wishlist persists using localStorage
